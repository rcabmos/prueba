1. ¿Cómo sabemos que un cuerpo se está moviendo?

El movimiento es el cambio de lugar o de posición con el tiempo de un cuerpo respecto a algo que tomamos como referencia.

A la hora de estudiar el movimiento de un cuerpo, el primer problema con que nos encontramos está en determinar, precisamente, si se está moviendo o no. Aunque la cuestión es, aparentemente, de fácil respuesta, realmente no es así.

1.1 Sistema de referencia

Para determinar el movimiento de un objeto hemos de tomar un sistema de referencia (que podemos considerar fijo) y observar la posición del cuerpo respecto de dicho sistema de referencia. Si su posición cambia con el tiempo, decimos que ese objeto se mueve respecto del sistema de referencia tomado.

 

En la imagen el sistema de referencia es el árbol, diremos que un objeto está en movimiento cuando cambien su posición respecto al árbol con el paso del tiempo.
